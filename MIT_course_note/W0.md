## Intro to Git/Github

git : a vision control system

### what git do?
git track changes(how? math!)
比如对于一个version1，新增了一行并commit，在MATH中可以表现为
```
+ content you add
@Line 4
```
如果继续又删除了一行，就是
```
- content you del
@ Line3
```

### some command line use
cp file1 file2 //备份
mv file1 file2 //重命名
rmdir - 删除文件夹，但是只能删除空文件夹
